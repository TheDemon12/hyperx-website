Features

  1. Note: This kernel only boots on custom roms for the OnePlus 3/3T (Won't boot on OOS/OB).
  2. Kernel has been based on AOSPA kernel source, thus battery life will be good. (PS. : kernel is not limited to AOSPA , will work on any custom rom)
  3. Linux 3.18.63
  4. CAF TAG: LA.UM.5.5.r1-05400-8x96.0
  5. Linaro-7.x Toolchain
  6. KCal V2 Color control
  7. Swipe2sleep gestures
  8. USB fast charging
  9. Implemented intermediate frequencies for LITTLE cluster instead of direct jump from 1.59 to 2.1 GHz.
  10. enabled ondemand & conservative governors.
  11. Custom governors include blu_active, elementalX, darkness
  12. Powersuspend Driver
  13. Adreno Idler
  14. Maple Iosched
  15. westwood TCP
  16. Some Wakelocks
  17. Boeffla Touch Key Driver [Use a terminal: echo 1 >> /sys/class/misc/btk_control/btkc_mode]
  18. qpnp-haptic: vibration controller
  19. Backlight Dimmer
